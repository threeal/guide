Use Read the Docs Theme in Sphinx¶

This section will explain a guide on setting up Read the Docs Sphinx theme in a Sphinx project.

See also

Read the Docs Sphinx Theme Installation on the official Read the Docs Sphinx theme documentation.

Theme Installation¶

Read the Docs Sphinx theme is available on pip as sphinx-rtd-theme package.

$ pip install sphinx-rtd-theme

Configure Theme in a Sphinx Project¶

Modify conf.py of the project by adding sphinx-rtd-theme in the extensions list and setting html_theme into sphinx_rtd_theme.

import sphinx_rtd_theme

extensions = [
  ...
  "sphinx_rtd_theme",
]

html_theme = "sphinx_rtd_theme"
