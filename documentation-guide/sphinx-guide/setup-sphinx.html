Setup Sphinx¶

This section will explain some of the options for installing Sphinx using APT and pip.

See also

Sphinx installation on the official Sphinx documentation.

Install Sphinx Using APT¶

Sphinx is available on APT as python3-sphinx package.

$ sudo apt install python3-sphinx

(Alternative) Install Sphinx Using pip¶

Alternatively, Sphinx is available on pip as sphinx package.

$ pip install -U sphinx
