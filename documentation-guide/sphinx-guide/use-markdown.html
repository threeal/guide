Enable Markdown in Sphinx¶

This section will explain a guide on enabling Markdown in a Sphinx project using MyST.

See also

  • Getting Started on MyST on the official MyST documentation.

  • Use Markdown in Sphinx on the official Sphinx documentation.

MyST Installation¶

MyST is available on pip as myst-parser package.

$ pip install -U myst-parser

Configure MyST in a Sphinx Project¶

Modify conf.py of the project by adding myst-parser in the extensions list and '.md' in the source_suffix.

import sphinx_rtd_theme

extensions = [
  ...
  "mystparser",
]

source_suffix = {
  '.rst': 'restructuredtext',
  '.md': 'markdown',
}
