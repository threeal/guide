Build Documentations Using Doxygen¶

This section will explain a guide on building documentations from a Doxygen project.

See also

Getting started on Doxygen on the official Doxygen documentation.

Build Documentations From a Doxygen Project¶

Run doxygen command to build documentations from a Doxygen project.

$ doxygen

Documentation results could be in HTML, LaTeX, RTF, or XML depend on the configuration file (Doxyfile) of the Doxygen project.
