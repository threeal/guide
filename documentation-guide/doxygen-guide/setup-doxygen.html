Setup a Doxygen Project¶

This section will explain a guide on setting up a Doxygen project.

See also

Doxygen installation and getting started on Doxygen on the official Doxygen documentation.

Doxygen Installation¶

Doxygen is available on APT as doxygen package.

$ apt install doxygen

Create a Doxygen Project¶

Use doxygen command with a -g option to create a configuration file (Doxyfile) for the Doxygen project.

$ doxygen -g

The resulted configuration file contains a lot of options with a description on it. But it could be simplified as follows:

PROJECT_NAME           = "My Project"
PROJECT_NUMBER         = "v0.1.0"
PROJECT_BRIEF          = "A brief description"
TAB_SIZE               = 2
INPUT                  = ./include
RECURSIVE              = YES
GENERATE_HTML          = YES
HTML_OUTPUT            = html
GENERATE_LATEX         = YES
LATEX_OUTPUT           = latex
